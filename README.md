# Bhanu Prasad - Professional Portfolio

Welcome to my personal portfolio! This repository contains the source code for my professional portfolio website, designed to showcase my projects, skills, and AI/full-stack development work.

## Features

- **Responsive Design**: Optimized for both desktop and mobile viewing.
- **Modern UI**: Clean and professional interface built with modern web technologies.
- **Project Showcase**: Highlights my latest work and technical capabilities.

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/gundabathinabhanuprasad/portfolio.git
   ```
2. Open `index.html` in your web browser.

## Deployment

This project is configured and ready to be deployed on Vercel.
